Pattern: Unexpected newline character

Issue: -

## Description

This rule enforces the type of new line characters.

## Configuration

-   Set `type` to `unix` to use UNIX-typed new line characters (`\n`), or `dos` to use DOS-typed new line characters (`\r\n`).
