Pattern: Style/IfUnlessModifier

Issue: -

## Description

Checks for if and unless statements that would fit on one line
if written as a modifier if/unless.
The maximum line length is configurable.

## Default configuration

Attribute | Value
--- | ---
MaxLineLength | 80
