Pattern: Style/WhileUntilModifier

Issue: -

## Description

Checks for while and until statements that would fit on one line
if written as a modifier while/until.
The maximum line length is configurable.

## Default configuration

Attribute | Value
--- | ---
MaxLineLength | 80
