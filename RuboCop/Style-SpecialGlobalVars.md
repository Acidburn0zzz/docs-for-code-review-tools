Pattern: Style/SpecialGlobalVars

Issue: -

## Description

This cop looks for uses of Perl-style global variables.

## Default configuration

Attribute | Value
--- | ---
EnforcedStyle | use_english_names
SupportedStyles | use_perl_names, use_english_names
