Pattern: Metrics/LineLength

Issue: -

## Description

This cop checks the length of lines in the source code.
The maximum length is configurable.

## Default configuration

Attribute | Value
--- | ---
Max | 80
AllowHeredoc | true
AllowURI | true
URISchemes | http, https
IgnoreCopDirectives | false
IgnoredPatterns |
