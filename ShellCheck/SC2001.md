Pattern: Use of `sed -e`

Issue: -

## Description

See if you can use `${variable//search/replace}` instead

Example of **incorrect** code:

```sh
string="stirng" ; echo "$string" | sed -e "s/ir/ri/"
```

Example of **correct** code:

```sh
string="stirng" ; echo "${string//ir/ri}"
```
